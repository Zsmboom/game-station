# 2048 Game Web Portal

一个使用 React + TypeScript + Vite 构建的 2048 游戏网站。

## 特性

- 🎮 经典 2048 游戏玩法
- 🎯 完整的游戏逻辑实现
- 💅 精美的 UI 设计
- 📱 响应式布局
- 🎨 原汁原味的 2048 配色方案
- 📖 详细的游戏规则说明

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- React Helmet Async

## 本地开发

```bash
# 克隆项目
git clone [你的仓库URL]

# 进入项目目录
cd web-game-portal

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 构建部署

```bash
# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```
